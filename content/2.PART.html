2.PART

第二部分前，請確認這三個資料夾是否都在，再進行操作

1.找到"putty"並打開資料夾的中"puttygen.exe"

點選load，找到id_rsa(若找不到請到右下角選擇"所有檔案"即可)

按"是"→Save private key→按"是"→輸入學號(存成.ppk檔)

2.打開資料夾中的"putty.exe"

(裡面若沒有存在"github.com"，在[Host Name]和[Save Sessions]打"github.com"按Save)

設定proxy(如使用自家網路或手機網路請選"None"這個選項)

下面那行是看你的網路連到哪一台主機 就打一樣的proxy進去

(我的是自家網路所以選擇"none")

打開SSH裡面的"Auth"

選擇你所存下金鑰檔案(.ppk)

回到session(滑到最上面那個)點選save再open

(成功就會出現下面這個)

3.在home裡面新增[.ssh資料夾]

ProxyCommand y:/putty/plink.exe github.com %h %p
  
Host github.com
    User git
    Port 22
    Hostname github.com
  
    TCPKeepAlive yes
    IdentitiesOnly yes

將上面的指令複製到白窗並另存一個"config"進去資料夾

4.在倉儲上新增一個金鑰

去到倉儲網頁，在右上角的頭像旁邊有個箭頭點開，在下方找到"Settings"

再從旁邊的目錄找到"SSH and GPG keys"

點New SSH key

到"tmp"找到"id_rsa.pub"並移到白窗中打開，複製裡面內容到Key來

{並確認自己學號}

按Add SSH key

第二部分完成，再來就是要進行如何上傳的部分了
