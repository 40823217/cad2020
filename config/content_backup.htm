製作表格

(此法也是學習於別人，若有錯誤請糾正!)

1.搜尋[試算表]，並點選第一個

2.選個人，點前往google試算表

3.點選右下的[建立新試算表]

4.當資料都輸入完時，找到[檔案]，往下看看到[發佈到網路]

5.選到[內嵌]，記得選擇[工作表1]，然後[複製原始碼]

6.接著到動態網站，到一個你要放表格的地方，然後進行編輯，之後找到[source code]點入

7.將剛剛那段[原始碼]貼入即可。

如何使用ssh

目的:利用ssh來commit資料

1.PART

首先要製作金鑰來做為驅動ssh的關鍵

1.點選資料夾中Portablegit→bin→sh.exe(之前有下載過的，如沒有請觀看老師的影片)

就會出現黑窗

輸入ssh-keygen -t rsa -b 4096 -C "40823217"(學號處請自行修改)

按enter→輸入/y/tmp/id_rsa→按2次enter

(在第3行應該會出現"y/tmp/id_rsa"這部分，之後再確認一下東西放哪就行)

2.開啟白窗，將"start.bat"移入，並加上這兩行文字

set GIT_HOME=%Disk%:\Portablegit\bin\
set GIT_SSH=%Disk%:\putty\plink.exe

(加入路徑的意思)

3.點選資料夾"cad2020"→找到git→將config移入白窗

(如若找不到，請點選檔案那行→找到檢視→在右邊找到"副檔名"這個選項，並打勾即可)

新增[remote "origin"]裡的url = git@github.com:40823217/cad2020.git

("#"記得加上)

之後將其save

第一部分完成，如有問題歡迎糾正

第二部分前請先下載"putty、putty_home、GitExtensions"

2.PART

第二部分前，請確認這三個資料夾是否都在，再進行操作

1.找到"putty"並打開資料夾的中"puttygen.exe"

點選load，找到id_rsa(若找不到請到右下角選擇"所有檔案"即可)

按"是"→Save private key→按"是"→輸入學號(存成.ppk檔)

2.打開資料夾中的"putty.exe"

(裡面若沒有存在"github.com"，在[Host Name]和[Save Sessions]打"github.com"按Save)

設定proxy(如使用自家網路或手機網路請選"None"這個選項)

下面那行是看你的網路連到哪一台主機 就打一樣的proxy進去

(我的是自家網路所以選擇"none")

打開SSH裡面的"Auth"

選擇你所存下金鑰檔案(.ppk)

回到session(滑到最上面那個)點選save再open

(成功就會出現下面這個)

3.在home裡面新增[.ssh資料夾]

ProxyCommand y:/putty/plink.exe github.com %h %p
  
Host github.com
    User git
    Port 22
    Hostname github.com
  
    TCPKeepAlive yes
    IdentitiesOnly yes

將上面的指令複製到白窗並另存一個"config"進去資料夾

4.在倉儲上新增一個金鑰

去到倉儲網頁，在右上角的頭像旁邊有個箭頭點開，在下方找到"Settings"

再從旁邊的目錄找到"SSH and GPG keys"

點New SSH key

到"tmp"找到"id_rsa.pub"並移到白窗中打開，複製裡面內容到Key來

{並確認自己學號}

按Add SSH key

第二部分完成，再來就是要進行如何上傳的部分了

3.PART

如何上傳

一:若完成第一部分跟第二部分的

直接進行"git add ."→"git commit -m "此更新的主旨"→"git push"

(此方法就不需打在打帳號密碼了)

二:使用"GitExtensions"

步驟如下→

1.點選[GitExtensions]中的"GitExtensions.exe"

然後修改裡面的設定

確認正確以後   點選"OK"

2.選擇"Create new repository"→點"Browse"→選自己的資料夾

然後create

點擊"tools"→"Git GUI"

先點選"Stage Changed"(等同於"git add ."的功能)

按是→按Continue(通常再按完"是"後會出現)

然後右下的位置打上這次的主旨(等於git commit -m "此更新的主旨")

然後就點擊Commit→Push→Push

成功!!

Develop

https://github.com/mdecourse/cmsimde 的開發, 可以在一個目錄中放入 cmsimde, 然後將 up_dir 中的內容放到與 cmsimde 目錄同位階的地方, 使用 command 進入 cmsimde 目錄, 執行 python wsgi.py, 就可以啟動, 以瀏覽器 https://localhost:9443 就可以連接, 以 admin 作為管理者密碼, 就可以登入維護內容.

cmsimde 的開發採用 Leo Editor, 開啟 cmsimde 目錄中的 cmsimde.leo 就可以進行程式修改, 結束後, 若要保留網際內容, 只要將 cmsimde 外部的內容倒回 up_dir 目錄中即可後續對 cmsimde 遠端倉儲進行改版.

init.py 位於  up_dir 目錄, 可以設定 site_title 與 uwsgi 等變數.
